# Sobre o projeto
- Programa que calcula o IMC (Índice de Massa Corpórea)

# Tecnologias utilizadas
- React Native
- Expo
- JS / TypeScript

# Como rodar
- Necessário ter o Node Js Instalado.
- No diretório, use o CMD e rode "npm install"
- Depois execute o comando "npm start"
